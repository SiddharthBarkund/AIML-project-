# Titanic Data Cleaning & Visualization Project

## Objective
Analyze Titanic dataset to understand survival patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Handled missing values
- Removed duplicates
- Created new features (family size, age group)
- Data visualization (graphs, heatmap)

## Key Insights
- Females survived more than males
- 1st class passengers had higher survival
- Higher fare → higher survival chance

## Output
- Cleaned dataset
- Visual insights
